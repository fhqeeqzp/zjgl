# TBGL - 工程造价管理系统

一个基于 PyQt5 的现代化工程造价管理系统，采用 QFluentWidgets 风格设计。

## 功能特性

### 用户认证
- 安全的用户登录系统
- 密码使用 SHA256 加密存储
- 支持回车键快速登录
- 用户名点击修改密码功能

### 项目管理
- 项目信息的新增、编辑、删除
- 项目类型：投标项目、施工项目
- 项目状态：投标中、进行中、已完成、已暂停
- 金额管理：投标金额、合同金额、实收金额、实付金额
- 日期管理：开始日期、竣工日期
- 附件管理：支持上传投标附件和施工附件
- 项目备注功能

### 数据展示
- 美观的数据表格（仿 QFluentWidgets 风格）
- 表格行高统一设置为 40px
- 项目名称列自动拉伸填充
- 支持右键菜单操作（编辑、删除）
- 现代化的表格样式设计

### 主题系统
- 支持深色/浅色主题切换
- 统一的样式管理系统
- 组件样式参数化配置
- 实时主题切换

### 界面特性
- 无边框现代化窗口设计
- 自定义标题栏
- 可拖拽移动窗口
- 圆角边框设计
- 响应式布局

## 技术栈

- **Python 3.14**
- **PyQt5** - GUI 框架
- **MySQL** - 数据库
- **SQLAlchemy** - ORM 框架
- **PyInstaller** - 打包工具

## 项目结构

```
.
├── components/         # UI 组件
│   ├── title_bar.py   # 自定义标题栏

│   ├── statusbar.py   # 状态栏
│   └── pages.py       # 页面组件
├── login/             # 登录模块
│   ├── ui/            # 登录界面
│   └── logic/         # 登录逻辑
├── XMGL/              # 项目管理模块
│   ├── ui/            # 项目界面
│   └── logic/         # 项目逻辑

```

## 样式规范

### 统一常量
- **圆角**：小 4px、中 6px、大 8px、窗口 10px
- **内边距**：小 5px、中 8px、大 10px
- **字体大小**：小 12px、正文 13px、标题 14px
- **边框宽度**：0.5px（细边框）

### 组件分类
- 按钮类：主要按钮、次要按钮、侧边栏按钮、关闭按钮
- 标签类：主要文字、次要文字、标题
- 输入框类：标准输入框、搜索框
- 卡片类：统计卡片、设置卡片
- 表格类：数据表格（仿 QFluentWidgets 风格）
